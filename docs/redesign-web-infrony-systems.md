# Rediseńo Web - INFRONY SYSTEMS

## 1) Objetivo General

Rediseńar el sitio web de **INFRONY SYSTEMS** con una experiencia moderna, tecnica y orientada a conversion para generar leads calificados.

Objetivo principal:

- Convertir visitantes en prospectos que soliciten una consultoria.

Publico objetivo:

- Pequenas y medianas empresas (PYMES)
- Startups
- Empresas que requieren automatizacion
- Organizaciones que quieren implementar IA
- Equipos que necesitan integracion de sistemas

---

## 2) Perfil de la Empresa

**INFRONY SYSTEMS** es una empresa de ingenieria tecnologica enfocada en:

- Automatizacion con Inteligencia Artificial
- Desarrollo de Software
- Integracion de Sistemas
- Infraestructura Cloud
- Sistemas Multi-Agente de IA

Producto insignia:

- **INFRONY ORCHESTRATOR**: plataforma de orquestacion multi-agente de IA para automatizar procesos empresariales conectando agentes, APIs y sistemas corporativos.

## 6) Estilo Visual Deseado

Direccion visual:

- Minimalista, moderno, limpio, tecnico
- Enfoque SaaS/AI enterprise
- Secciones claras con jerarquia fuerte
- Uso de gradientes suaves y brillos sutiles

Elementos visuales:

- Tarjetas con hover elevation
- Transiciones suaves
- Fondos con grid animado
- Nodos flotantes de IA
- Diagramas de automatizacion y pipelines

Animaciones recomendadas:

- Fade-in por seccion
- Stagger en cards
- Hover interactivo en componentes clave
- Diagrama animado de flujo de datos/agentes
